# ATM Simulation System

A simple Java project that simulates basic ATM operations such as:
- Account login
- Balance check
- Deposit and withdrawal
- Transaction history

## Technologies Used
- Java
- IntelliJ IDEA
- OOP Concepts

## How to Run
1. Clone the repository
2. Open it in IntelliJ IDEA
3. Run the Main.java file inside src/ATM
